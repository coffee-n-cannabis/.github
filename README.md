# coffee-n-cannabis / .github

Shared defaults for the coffee-n-cannabis GitHub account. The files here are reference copies that other repos in this account can adopt as-is or as templates.

## What's here

- **`SECURITY.md`** — how to report a security issue in anything we ship.
- **`CODE_OF_CONDUCT.md`** — what we expect from people we work with, or who interact with our work in public.
- **`.gitignore`** — standard exclusions: editor swap files, OS artifacts, `.env`, build output.
- **`README.md`** — this file.

## What's not here

- **No `FUNDING.yml`.** We don't run a sponsorship page. Donations welcome at `info@coffee-n-cannabis.com`.
- **No issue or PR templates yet.** We'll add them when a public repo here gets enough traffic to need them.
- **No roadmaps, no manifestos, no corporate niceties.** See the [profile README](https://github.com/coffee-n-cannabis) for the working method.
